how to play?


        

        

            griddle is a simple game where you find a word by picking one letter from each row of the grid.
        


        

            when you have made a valid word your score will be displayed.
        


        

            there are two ways to play the game, either aiming for the highest possible score, or the lowest possible score.
        


        

            the score is calculated based on how many other valid words start with the same letters as the word you selected.
        


        

            for example for the 4x4 grid 

            
fmrt
atui
ntls
tpek
            


        


        

            there are 53 possible words, the highest possible score is 35 and the lowest possible score is 11.
        


        

            the highest scoring word is mine as there are 22 valid words that start with m, 9 valid words that start with mi, 3 valid words that start with min, and one valid word that starts with mint
        


        
22 + 9 + 3 + 1 = 35



        

            the lowest scoring word is fate as there are 7 valid words that start with f, 2 valid words that start with fa, 1 valid word that starts with fat, and one valid word that starts with fate
        


        
7 + 2 + 1 + 1 = 11


        
it is possible for there to be multiple words that match the highest and lowest scores
